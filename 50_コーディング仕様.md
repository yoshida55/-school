# ランディングページ（英会話スクール）コーディング仕様

## 基本設定

| 項目 | 値 |
|------|-----|
| フォント | 游ゴシック |
| コンテンツ幅 | 860px |
| ブレイクポイント | 900px |

---

## 各セクション仕様

### Header
- ロゴ：左上に配置（白色）
- ナビゲーション：右側に配置
  - BBBが選ばれる理由
  - 受講生の声
  - スクールの概要
- 「無料体験に申し込む」ボタン：右端に配置
- **SP時：**ハンバーガーメニューでフェード全画面表示
  - 開閉時にメニューアイコンが×に変化

### メインビジュアル
- 幅：全幅（100%）
- 高さ：画面下まで（`100vh`）
- 3枚の画像をCSSアニメーションでフェード切り替え
  - 切り替え時間：5～7秒
  - アニメーション：`animation` プロパティ使用
- テキスト
  - 「話して学ぼう！」（小見出し）
  - 「BBB英会話スクール」（大見出し）
- 「無料体験はこちら」ボタン
  - 画像に重なるように配置（`position` 使用）
  - ホバー時：ボタンのサイズが少し大きくなる（`transform: scale()`）

### BBBが選ばれる理由
- セクションタイトル
  - 「BBBが選ばれる理由」
  - 英字と日本語の2段構成
- 2カラム構成
  - 左：「オンライン対応」アイコン + テキスト
  - 右：「講師はネイティブ」アイコン + テキスト
- **アニメーション：**
  - 左カラム：左からスライド表示
  - 右カラム：右からスライド表示
  - 実行タイミング：要素が画面下に表示されたとき（jQueryプラグイン「inview」使用）
  - CSSアニメーション：`@keyframes` で定義
- **SP時：**1カラムで縦並び

### 受講生の声
- セクションタイトル：「受講生の声」
- 3カラム構成
  - 各カラム：アイコン画像 + 名前・職業 + ふきだし（テキスト）
- **ふきだし**
  - CSSで作成（`::before` または `::after` 疑似要素）
  - PC時：ふきだしの三角は上向き
  - SP時：ふきだしの三角は左向き（向きを切り替える）
- **アニメーション：**
  - 非表示状態（`opacity: 0` / `transform: scale(0)`）から、ふわっと表示
  - 少しずつ大きくなる（`transform: scale(1)`）
  - 実行タイミング：要素が画面下に表示されたとき（jQueryプラグイン「inview」使用）
- **SP時：**1カラムで縦並び

### スクールの概要
- セクションタイトル：「スクールの概要」
- 背景：ドット柄画像
  - **アニメーション：**ドット柄を右上から左下へ繰り返し流す
  - `background-position` をアニメーション
  - `@keyframes` で無限ループ
- 4カラム構成
  - レッスン内容 LESSON
  - 料金プラン PRICE
  - 講師のご紹介 TEACHER
  - BBBのQ&A Q&A
- 各カラム：英字タイトル + 日本語タイトル + テキスト
- **SP時：**2カラム（2行）で表示

### CTA（無料体験エリア）
- 背景：画像（全幅）
- テキスト
  - 「まずは無料で、BBBの英会話を試してみませんか？」
  - 「今なら初月のレッスンを特別価格で受講できる割引クーポンをプレゼント！」
- 「無料体験に申し込む」ボタン
  - 通常時：赤背景 + 白文字
  - ホバー時：白背景 + 赤枠線 + 赤文字（`background`, `border`, `color` を変化）

### Footer
- 背景：濃いグレー
- ナビゲーションメニュー（4カラム）
  - コース一覧
  - 講師紹介
  - 会社情報
  - BBBについて
- コピーライト：「© BBB English School」
- **SP時：**2カラム（2行）で表示

## JavaScript（jQuery）仕様

### 使用プラグイン
- **jquery.inview.min.js**
  - 要素が画面下に表示されたタイミングでアニメーション実行
  - 配布元：https://github.com/protonet/jquery.inview

### 実装内容
1. ハンバーガーメニューの開閉（フェード表示）
2. メニューアイコンの×への切り替え
3. inviewプラグインで下記アニメーションを実行
   - 「BBBが選ばれる理由」のスライド
   - 「受講生の声」のふわっと表示

---

## CSSアニメーション仕様

### メインビジュアル画像の切り替え
```css
@keyframes fade {
  /* 3枚の画像を順番にフェード表示 */
  /* 各画像の表示時間を調整（5～7秒） */
}
```

### BBBが選ばれる理由（スライド）
```css
@keyframes slideFromLeft {
  /* 左からスライド */
}

@keyframes slideFromRight {
  /* 右からスライド */
}
```

### 受講生の声（ふわっと表示）
```css
@keyframes fadeInScale {
  /* opacity: 0 → 1 */
  /* transform: scale(0) → scale(1) */
}
```

### スクールの概要（ドット柄移動）
```css
@keyframes moveBackground {
  /* background-position を右上→左下へ移動 */
  /* 無限ループ（animation-iteration-count: infinite） */
}
```
